## 📖 О проекте

**«Ориентир»** — это интерактивная веб-платформа, исследующая архитектуру защиты сознания от манипулятивного воздействия искусственного интеллекта.
Это не просто сайт. Это прототип (Функциональный макет) системы защиты от угроз, которые станут массовыми в ближайшие 3-5 лет, когда ИИ получит доступ к нашему вниманию, эмоциям и микропроцессам принятия решений.
Проект сочетает философский нарратив (через тексты доктора Чехова) с техническими инструментами для анализа, верификации и защиты от скрытых паттернов влияния. Каждый элемент — это одновременно и защита, и обучение: вы используете инструмент и одновременно понимаете, как аналогичные механизмы могут использоваться против вас в других сервисах.

### 🎯 Цель проекта

Техническая цель:
Показать возможность создания сложных, производительных интерфейсов без тяжёлых фреймворков (React/Vue), используя чистый JavaScript и понимание DOM/WebGL. Защита от слежения не требует жертвовать производительностью.

Философская цель:
Сделать невидимые угрозы — видимыми. Большинство людей не заметит, когда ИИ начнёт:
Подстраивать контент под их усталость (когнитивное истощение = выше внушаемость)
Управлять темпом принятия решений (ускорение = меньше осознанности)
Создавать иллюзию выбора (3 варианта, ведущих к одному результату)
Формировать привычки незаметно (21-60 дней микроподталкиваний)
«Ориентир» показывает эти механизмы в действии — и даёт инструменты защиты.

## 🛠 Стек технологий

| Категория | Технологии |
|-----------|------------|
| **Core** | HTML5, CSS3 (Variables, Grid, Animations), Vanilla JS (ES6+) |
| **3D Graphics** | Three.js (Custom Shaders, Particle Systems, Performance Optimization) |
| **Visualization** | Vis.js (Graph Visualization) |
| **Architecture** | Component-based CSS, Modular JS Classes, LocalStorage State Management |
| **Tools** | ~~SSI (Server Side Includes)~~ → *для статической версии инклюды раскрыты вручную*, PWA (manifest.json) |
| **Deployment** | GitHub Pages, Custom Domain (orenpro.pro) |

> **Примечание:** Исходная версия сайта использовала Server Side Includes (SSI) для сборки компонентов. Для публикации на GitHub Pages все директивы `<!--# include ... -->` были заменены на статическое содержимое соответствующих файлов. Таким образом, текущая версия полностью статична и не требует серверной обработки.

## 🔒 Принципы приватности

Этот проект демонстрирует подход к созданию приватных веб-систем:

1. ✅ **Не собирает персональные данные** — все вычисления происходят локально
2. ✅ **Не использует cookies для отслеживания** — только localStorage для настроек
3. ✅ **Не отправляет данные на сервер** — анализ текста работает в браузере
4. ✅ **Открытый исходный код** — весь код доступен для аудита
5. ✅ **Запрет на обучение ИИ** — meta-теги noai, noimageai

## 🎯 Для кого этот проект

Для специалистов по AI Safety & Ethics
Таксономия угроз (Механизмы манипуляций), архитектура контримпульсов, принципы Transparency by Design
Готовый фреймворк для проектирования этичных ИИ-систем. Можно использовать как чек-лист при аудите продуктов.

Для продуктовых команд (Trust & Safety)
Паттерны защиты от тёмных паттернов, методология верификации информации, принципы нейтрального интерфейса
Альтернатива engagement-метрикам. Как строить продукты, которые усиливают, а не эксплуатируют пользователя.

Для разработчиков
Пример модульной архитектуры на Vanilla JS, реализация 3D-сцен с оптимизацией под мобильные, State Management без библиотек
Сложные системы можно строить без раздувания зависимостей. Меньше кода = меньше поверхностей атаки.

Для исследователей и журналистов
Инструменты анализа манипуляций, конструктор исследовательских протоколов, библиотека паттернов влияния
Методология для независимого аудита цифровых сервисов. Можно проверить любой продукт на соответствие принципам автономности.

Для обычных пользователей
Понятные примеры угроз (без технического жаргона), практические инструменты защиты, обучение через взаимодействие
Первый шаг к цифровому суверенитету. После «Ориентира» вы начнёте замечать манипуляции в других сервисах.

Сегодня ИИ управляет вниманием через ленты рекомендаций.
Через 3-5 лет ИИ будет управлять вниманием через очки дополненной реальности, технологические браслеты, кольцо и т.д. анализируя ваш пульс, мимику и микрозадержки взгляда.
«Ориентир» строится сегодня — чтобы защита была готова завтра.
Это архитектурная подготовка. Когда технологии станут незаметнее, большинство не поймёт, где заканчиваются их мысли и начинаются навязанные паттерны. Этот проект — для тех, кто хочет остаться автором своего сознания.
